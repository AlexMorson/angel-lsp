# Freatures

- Syntax Highlight for AngelScript
- Go to Definition
- Peek References
- Symbol Rename
- Symbol Completions
- Type Checking

Note: Still in the early stages of development, many features of AngelScript are not fully supported

OpenSiv3D Support
